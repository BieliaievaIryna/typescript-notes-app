# Notes Management App

Додаток для управління нотатками з використанням принципів ООП, патерну DTO та декораторів у TypeScript.

## Функціонал

- **Додавання нотаток**  
  Нові нотатки завжди створюються як дефолтні (`DefaultNote`).  

- **Редагування та видалення нотаток**  
  - Якщо нотатка має `type: confirmation`, для редагування або видалення вимагається підтвердження (`ConfirmationNote`).  
  - Дефолтні нотатки редагуються та видаляються без підтвердження.  

- **Позначення нотатки як виконаної**  

- **Статистика**  
  Показує загальну кількість нотаток та кількість невиконаних.  

- **Пошук та сортування**  
  - Пошук за назвою або змістом.  
  - Сортування за статусом виконання або датою створення.  

## Робота з даними

- Вхідні дані приходять у форматі **snake_case**.  
- Внутрішня логіка використовує **camelCase**.  
- Типи `MapToCamelCaseDomain` та `MapToSnakeCaseDTO` автоматично трансформують ключі між форматами.  

## Декоратори

- `@SanitizeInput` – видаляє пробіли на початку та кінці назви та змісту нотатки.  
- `@ValidateNotEmpty` – перевіряє, що назва та зміст не порожні.  
- `@AutoUpdateTimestamp` – автоматично оновлює `updatedAt` при редагуванні нотатки.  

## Технології

- TypeScript  
- OOP, Наслідування класів  
- Декоратори  
- DTO (Data Transfer Object)  

## Приклад використання

```ts
const notesList = new NotesList(mockServerResponse);

notesList.addNote({
  noteId: '4',
  noteTitle: 'Нова нотатка',
  noteContent: 'Текст нотатки',
  isCompleted: false,
  type: 'default',
});

notesList.editNote('2', { noteContent: 'Оновлений текст' });
notesList.deleteNote('3');

console.log(notesList.getStats());
```
